
# Restore
E-Commerce Cart Website
---

A full-stack e-commerce cart application built to demonstrate real-world shopping cart functionality, clean UI design, and scalable backend architecture using modern web technologies.

---

## Features

- Product listing and cart management  
- Add, remove, and update cart items in real time  
- Centralized and predictable state management using Redux Toolkit  
- Secure RESTful APIs for frontendâ€“backend communication  
- Responsive UI optimized for desktop and mobile devices
